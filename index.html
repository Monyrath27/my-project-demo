About

TravelVista is a simple and interactive website designed to help users explore countries, cultures, and travel tips through a clear and visual experience.

The site was built using HTML, CSS, and animations to make learning about the world engaging and beginner-friendly.

TravelVista was created by Monyrath during the Sister of Code program, where I learned how to design, structure, and build a website from scratch.
